## Расширение для Google Chrome.
Разработало для использования на twitch.tv. Добавляет пользователям в чате настраиваемые подписи.
Настройки сохраняются в формате json, в синхронизируемом хранилище chrome, что (я надеюсь) позволяет использовать расширение на разных ПК, где выполнен вход в аккаунт Google (не тестировалось).

### Основные сценарии использования:
1) Записать какого-либо пользователя по реальному имени или по имени, как к нему обращаться.
2) Подписать аккаунты пользователей-мультоводов, чтобы быстро понимать кто это.
3) Подписать друзей или моральных уродов

### Установка:
Скачать zip-архив с исходниками. Распаковать в папку, где оно будет храниться (где Вам нравится).
В браузере перейти в настройки расширений chrome://extensions/ включить переключатель "Режим разработчика" в правом верхнем углу.
Там же, в браузере нажать "Установить распакованное расширение", указать папку, куда было распаковано расширение.
Если были открытые страницы twitch - обновить их для запуска расширения.

### Настройка:
При клике на иконку расширения появляется окно с одним полем ввода и одной кнопкой "Сохранить".
При открытии окна настроек в поле ввода вычитываются настройки, сохраненные ранее в хранилище Google Chrome. При нажатии кнопки "Сохранить" - настройки сохраняются в хранилище Google Chrome.
Формат настроек - JSON-объект, в котором ключи атрибутов соответствуют никнеймам пользователей, а значения - подписям, которые будут добавляться.
Пример (есть в скачиваемом архиве, example.json):
```
  {
    "user1": "из counter-strike",
    "user2": "Придурок",
    "user3": "Иван",
    "user4": "Иван (второй акк)",
    "user5": "клиподел"
  }
```
